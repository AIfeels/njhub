# Safe City Demo Video Creation Guide

## Quick Video Creation Options:

### Option 1: Use AI Video Generators
1. **Synthesia.io** - AI avatar with Hindi voiceover
2. **Loom** - Screen recording with voiceover
3. **Canva Video** - Template-based video creation
4. **InVideo** - AI-powered video maker with Hindi support

### Option 2: Simple Screen Recording
1. Use **OBS Studio** (free) or **Camtasia**
2. Record Safe City software interface
3. Add Hindi voiceover using **Audacity**
4. Use **DaVinci Resolve** (free) for editing

### Option 3: PowerPoint Video
1. Create slides with Safe City features
2. Add animations and transitions
3. Record slideshow with Hindi narration
4. Export as MP4
